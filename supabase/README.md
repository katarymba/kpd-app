# База данных Supabase

## Применить миграции

1. Зайди на https://supabase.com/dashboard
2. Выбери проект
3. Перейди в раздел SQL Editor
4. Открой файл `supabase/migrations/001_initial_schema.sql`
5. Скопируй содержимое и выполни в SQL Editor
6. Нажми Run

## Настройки Auth

В Supabase Dashboard → Authentication → Settings:
- Disable email confirmations (для удобства тестирования)
- Site URL: https://kpd-app.vercel.app
